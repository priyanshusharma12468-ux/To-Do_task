# 📝 Todo List Application

A simple Todo List web application built using **HTML, CSS, and JavaScript**. It allows users to manage daily tasks with features like adding, editing, deleting, marking tasks as completed, and saving tasks using Local Storage.

## 🚀 Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Save tasks using Local Storage
* Automatically load saved tasks after page refresh
* Simple and responsive user interface

## 📂 Project Structure

```text
todo-list/
│── index.html
│── style.css
│── script.js
│── README.md
```

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Start adding and managing your tasks.
